# FlowDesk

FlowDesk 是一个面向企业内部协作场景的工单与审批协同平台，支持工单创建、流转处理、审批操作、日志留痕、数据统计与导出。

## 项目价值

- 贴近真实公司业务系统，不是简单 CRUD
- 能体现状态流转、角色权限、审批链路、日志追踪
- 很适合作为前端 / 全栈岗位的业务型作品

## MVP 模块

- 工单中心
- 审批台
- 状态流转
- 操作日志
- 统计看板
- 导出记录

## 技术栈

- Frontend: Vue 3 + Vite + Vue Router + Element Plus
- Backend: Node.js + Express
- Database: MySQL
