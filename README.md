# Work Portfolio

这是我的项目作品集总仓库，主要收录了几类不同方向的完整项目，用于展示前端、全栈、AI 协同应用和业务流程系统能力。

## GitHub

- GitHub 主页: [https://github.com/zengyi200413](https://github.com/zengyi200413)

## 项目列表

### 1. CareerPilot AI

- 方向：AI 求职材料协同平台
- 亮点：岗位 JD 解析、简历文件上传、PDF 自动提取、匹配分析、项目亮点改写、模拟面试题、人工审核与导出
- 独立仓库：[careerpilot-ai](https://github.com/zengyi200413/careerpilot-ai)

### 2. DocPilot AI

- 方向：AI 文档协同平台
- 亮点：文档上传、文本解析、摘要生成、行动项提取、PRD 初稿、审核台、导出记录
- 独立仓库：[docpilot-ai](https://github.com/zengyi200413/docpilot-ai)

### 3. FlowDesk

- 方向：工单与审批协同平台
- 亮点：工单提交、审批流转、状态变更、日志留痕、统计报表、导出记录
- 独立仓库：[FlowDesk](https://github.com/zengyi200413/FlowDesk)

### 4. Blog Admin

- 方向：全栈博客与后台管理系统
- 亮点：后台管理、文章分类标签、评论互动、前台展示、读者登录与个人中心
- 独立仓库：[blog-admin](https://github.com/zengyi200413/blog-admin)

## 适合简历展示的重点项目

- CareerPilot AI：更适合突出 AI 协同应用开发能力
- FlowDesk：更适合突出真实业务流程建模能力
- DocPilot AI：更适合补充文档工作流与审核机制能力
